User Story

Adding specializations for an existing Little Library

As a Little Library Steward, I want to add a specializations to my Little Library including technical books for my interests and children's books for my children.

User Interaction Flow

Pre-Condition:

Stanley is registered as a Steward, has logged into the site and is on the landing page.

Post-Condition

Stanley has added two specializations to his Little Library and those specializations appear on the map and can be searched for by other users.

User Interaction Flow

  • Stanley scans the landing page and locates the sidebar nav button on the right side of the main element (map) for the "Steward" section and clicks it.
  • Little Library redirects Loco to the "Stewards" page.
  • Stanley scans the "Steward" page and finds the Little Library Maintenance menu and selects the "Add specialization to a Little Library" button and clicks it.
  • Little Library queries the database and returns a modal listing Stanley's Little Libraries with the location of each Little Library associated with Stanley.
  • Stanley clicks the link for the Little Library he wants to add the specialization to.
  • Little Library presents the Little Library maintenance modal to Stanley where he is able to edit his Little Library listing, including specializations.
  • Stanley scans the modal and locates the section where he can add up to three specializations to his Little Library from drop down menus.
    • Stanley chooses "Text Books & Technical Books" from the first menu.
    • Stanley chooses "Children's Books (Ages 1- 10)" from the second menu.
    • Stanley clicks "Submit" to complete the edit to this listing.
  • Little Library reads the input from the modal when "Submit" is clicked and updates the database.
  • Little Library presents Stanley with an informational modal stating "Edits to Little Library complete.
