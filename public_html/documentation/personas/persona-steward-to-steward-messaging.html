User Story & Inter Flow - Steward To Steward Messaging

User Story

As new steward user, I want to message other LL stewards to ask about LL setup, what books are most needed in our area and see if any steward has contacted the local library.

Interaction Flow

Use case:

Martha Mess is a homemaker that books and reading and wants to set up a LL in her neighborhood. She has been set up as a steward and is ready to setup her LL. She wants to contact other stewards to get advise on setting the LL up, getting books for it and if they have contacted the local library about posting events and LL locations.

Precondition

Martha Mess is on her iPhone and has signed into her steward account. She is on the landing/home page.

Postcondition

Martha Mess has successfully sent a message to other stewards asking for the information she wants.

Required Steps:

  • User scans the landing page and locates the sidebar nav button for the "Steward" section that is on the right side of the map and clicks it.
  • The app redirects the user to the "Stewards" page.
  • The user scans the "Stewards" page and finds the "Steward Message Board" list, locates the button to add a message and clicks it.
  • LL app opens a messaging form with her steward information preloaded. There is a text field and a submit button.
  • User enters the message to get info on setting up her LL, getting books, if the local library has any coming events and if the local library has a list of the nearby LL and then clicks the submit button.
  • The app displays a confirmation the message has been added for 5 seconds then displays the "Stewards" page.
  • The user verifies the message is showing on the message board.
