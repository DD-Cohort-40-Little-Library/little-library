User Story & Inter Flow - User Meet Friends

User Story

As a regular user of the LL app David Kiplinger wants to set up a casual meet up for his friends since he will be in a specific location for the day.

Interaction Flow

Use case:

David Kiplinger is planning on spending the day in a specific location while his car is getting repaired. He thought it would be fun to meet his friends in at a LL for coffee, books and fun.

Pre-Condition:

David has an established account and is logged in.

Post-Condition:

David knows the general location where he will be, date, time and friends he wants to connect with.

Required Steps:

Interaction Flow Option 1

  • User utilizes the map feature and searches out the general location which he will be at.
  • Map interface populates with all the LL in the search distance criteria he selected. (within 25 mi)
  • User uses the ZOOM function to pinpoint the exact LL that will serve as his meet up point.
  • User taps/clicks to select exact LL location
  • Map interface launches a pop-up with LL specific information, including "Check-In" and "Report Problem" and "Meet w/ Friends" buttons.
  • User chooses and taps/clicks "Meet w/ Friends" button
  • Interface pop-up window launches with Meet Up invite form. This includes date, time, message and radio select friends and invite all options.
  • User clicks "Send Invite" button.
  • Confirmation of "Send Invite" pops-up occurs. Pop-up includes a notification for this event on/off option
  • User selects notification on button and closes pop-up
