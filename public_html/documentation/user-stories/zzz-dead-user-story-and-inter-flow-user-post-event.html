User Story & Inter Flow - User Post Event

User Story

As an intermediate technology user, I want to post an upcoming event at the library where I work.

Interaction Flow

Use case:

Dedre Kiplinger is a librarian and has a live reading event scheduled at her library next week. Dedre has an active LL account and has been telling everyone coming into the library about the live event. She does not have any fliers about the event and has been telling everyone to sign up for the LL app to find the event that will have the schedule of live reading times. Dedre now needs to post the event on the LL app.

Precondition

Dedre Kiplinger is on her iPhone and has signed into the LL app and is looking at the landing page.

Postcondition

Dedre Kiplinger has successfully posted the live reading event the will be at her library next week.

Required Steps:

  • User scans the home page and finds the list of "Coming Events" section. The "Add Coming Event" link is located and clicked.
  • The "Add Coming Event" popup/page is opened with fields for "Event Name", "Event Date", "Event Time(s)" and "Event Description" fields available.
  • User enters relevant information in each field, but enters the date as text and numbers instead of the required format of mm/dd/yyyy.
  • The app pops up a reminder message that dates must be entered in the format of mm/dd/yyyy and clears the date field.
  • User enters the correct date in the correct format and clicks the submit button.
  • The app confirms the event has been saved and uploaded to the "Coming Events" list and the entry popup closes and the home page is shown."
  • The user scans the "Coming Events" list and confirms the event she entered is there.
