User Story & Inter Flow - User Find Event

User Story

As a consistent user of the LL app and an intermediate technology user, I would like to use the app to check in to the LL I am currently visiting.

Interaction Flow

Use case:

Charles Check is a late career teaching professional who has been using the application consistently to find LLs and interact with its community. Charles had been visiting family out of state and stumbles upon a unfamiliar established LL and wants to "Check In" on the app. He has an established account and is ready to use the app to check in to the LL he is now visiting.

Pre-Condition:

Charles Check is looking at the LL home page on his phone and is familiar with the check in process as he uses the feature often.

Post-Condition:

Charles Check is currently physically at the LL he would like to check in to.

Required Steps:

  • User uses the map search function locate the specific LL by either zip, city/state, or "use my location" phone function.
  • Map interface populates with all the LL in the search distance criteria he selected. (within 25 mi)
  • User uses the ZOOM function to pinpoint the exact LL he is currently at.
  • User taps/clicks to select exact LL
  • Map interface launches a pop-up with LL specific information, including "Check-In" and "Report Problem" buttons.
  • User chooses and taps/clicks "Check-In" button
  • Confirmation of "Check-In" pops-up occurs.
  • User closes pop-up
