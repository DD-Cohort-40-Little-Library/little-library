User Story: User Find Friends - Franchesca Friend

User Story

As a new user, I want to find and connect with friends, make new friends and find local events and the little libraries near me.

User Workflow

Assumptions:

User has the app/site open.

Workflow

  • User clicks on the feature to find friends.
  • App provides list of friends.
  • User scans for a friend to connect with and clicks on their name.
  • App opens message field for the friend.
