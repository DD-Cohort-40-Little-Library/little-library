Developer User Stories by Entity

User

As a developer, I will know that sign-up function works properly when I am able to make several unique accounts and receive and error when it is a duplicate account.

As a developer, I will know that sign-in is working when I can sign into an existing account or will receive an error if I try to sign in but my credentials do not match an existing account.

As a developer, I will know that the sign-out function is working when a user selects the sign-out option (or has been idle on the site for a set period of time?).

As a developer, I will know that the edit account function is working when I am able to edit the appropriate fields of a user account and the account reflects those changes.

As a developer, I will know that the upload avatar function is working when I can upload an avatar to an account and use the avatar on that account.

Library

As a developer, I will know that the register library function is working when I am able to add a library with all or the minimum fields completed and will receive an error if the library is a duplicate or is missing required fields.

As a developer, I will know that the modify library function is working when I am able to modify a library listing that is associated with a specific user account.

As a developer, I will know that the follow library function is working when I am able to follow or unfollow a library

As a developer, I will know that the upload library photo function is working when I can upload a photo of the library and use it for the library that is associated with a specific user account.

As a developer, I will know that the library map is working when the landing page opens and all libraries show on the map in the appropriate manner.

As a developer, I will know that the library map modal is working when I click on the map pin and the appropriate library shows in the modal.

As a developer, I will know that the library details function is working when the library detail page opens for the selected library.

As a developer, I will know that the select library function is working when I can view a specific library that is selected.

As a developer, I will know that the select library by specialization function is working when I can select a library specialization or specializations from the landing page selectors and have just those libraries with that specialization appear on the map.

As a developer, I will know that the add specialization function is working when I am able to add a specialization to library that is associated with a specific user account.
